# vim-spef

Vim syntax highlighting for SPEF (Standard Parasitic Exchange Format) files.
SPEF is defined by IEEE Standard 1481-1999 for representing parasitic RC data extracted from IC layouts.
## Installation

Copy 
`spef.vim`
 to 
`~/.vim/syntax/`

Copy 
`spef_ftdetect.vim`
 to 
`~/.vim/ftdetect/spef.vim`

##Usage

Open any `.spef` file - to apply automatic highlighting 

Manually:
```
vim
:set filetype=spef
```
